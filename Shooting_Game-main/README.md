# Shooting Game

A Unity-based first-person shooting game featuring realistic weapon mechanics, zombie enemies, and destructible targets.

## Key Features

- **First-Person Controller**: Smooth movement with jumping and gravity simulation.
- **Advanced Weapon System**:
  - **Weapons**: Includes **Pistol 1911** and **M4** Assault Rifle.
  - **Shooting Modes**: Support for **Single**, **Burst**, and **Auto** fire.
  - **Realism**: Realistic reloading, ammo management, recoil animation, and bullet spread.
- **Enemy AI (Zombies)**:
  - Zombies with Patrol, Chase, and Attack states.
  - Damage system with health and death animations.
- **Interaction System**: 
  - Look at weapons to highlight them.
  - Pick up weapons from the environment.
- **Destructible Environment**: Interactive bottle targets that shatter upon impact.
- **Audio & Visuals**: Immersive sound effects, muzzle flashes, and bullet impact particles.

## Controls

| Action | Input |
| :--- | :--- |
| **Move** | `W`, `A`, `S`, `D` |
| **Jump** | `Space` |
| **Shoot** | `Left Mouse Button` |
| **Reload** | `R` |
| **Interact / Pickup** | `F` |
| **Look** | `Mouse` |

## Setup Instructions

1.  Open the project in **Unity**.
2.  Navigate to the `Assets/Scenes` directory.
3.  Open the main game scene.
4.  Press **Play** to start hunting zombies!
